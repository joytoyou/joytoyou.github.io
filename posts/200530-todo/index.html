블로그에 추가해야 할 기능들

2020/05/30

따로 블로그를 위한 static site generator를 쓰지 않고 맨 처음부터 그냥 내가 필요한 기능만 추가해가면서 만들고 있기 때문에 추가해야 할 기능들이 엄청엄청엄청 많다. 그래서 리스트를 만들고 해결되면 구현 방법을 하나씩 추가하기로!

  • Markdown에 checkbox extension 추가
    • 나는 checkbox가 마크다운 기본 기능인 줄 알았는데 왜 안 되나 싶어서 찾아봤더니 따로 기능을 추가해야줘야 하는 거였네 markdown-checklist
    • 근데 checkbox 나오게 할 땐 앞에 점 사라졌으면 좋겠어~~~~
      ul.checklist { list-style-type: none; }
      li input { margin-left: -20px; }
      CSS에서 점을 안 보이게 하고 그 자리에 checkbox를 밀어서 해결😁
    • 근데 Github Markdown처럼 쓰려면 넘어야 할 산이 많구나......
  • HTML rendering😔
    • Jinja2 template에서 block 안에 내용을 넣어줄 때 생기는 이상한 newline과 indent 불일치는 여기를 보고 해결
    • Markdown이 rendering하는 HTML의 indent는 이걸로 해결
  • 뒤로 가기 버튼 추가
    • 대충 추가는 했는데 지금은 그냥 블로그 메인 화면으로 보내고 있다 (어차피 글 목록 & 포스트 밖에 없으니까) 본래의 뒤로 가기 기능을 수행하게 하는건 블로그가 조금 더 복잡해지면 하기로!
  • Twitter와 블로그 연동 (조금 부끄러워서 비밀계정에 하고 있는 중)
    • https://github.com/bear/python-twitter
    • 지금은 PostUpdate로 블로그의 새 글 알림을 주는 용도로만 사용하고 있지만 아마 더 많은 일을 할 수 있겠지!
  • DISQUS로 댓글 기능 추가
  • 포스트 metadata 독립적으로 설정
  • Category / Tags
  • Code Block 예쁘게 하기😊
  • 스크롤 할 때 왼쪽 사이드바는 움직이지 않게 하기
    • 이 부분 여러 시도를 해봤는데 계속 내가 원하는 대로 되지 않아서 살펴보니 원래 테마에서 내 생각과는 다르게 구현이 되어 있었다. 그래서 내가 원하는대로 동작하게 하려면 그냥 내가 원래 하던 방식대로 다시 만들어야 할 것 같은데... 귀찮다...
    • 일단 overscroll-behavior: none; 으로 해결
  • Cache Invalidation
  • File modification detection
